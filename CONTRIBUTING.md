# Contributing to Awesome AI Companions

Thank you for contributing! This list is meant to be a comprehensive, up-to-date resource for anyone interested in AI companion apps and platforms.

## How to Contribute

### Adding a New App/Tool

1. Fork this repository
2. Add your entry to the appropriate section in `README.md`
3. Maintain alphabetical order within each table
4. Follow the format exactly (see below)
5. Open a Pull Request with a clear description

### Entry Format

For app tables:
```markdown
| [App Name](https://app-url.com) | Key feature 1, key feature 2, key feature 3 | $X/mo or Freemium | iOS, Android, Web |
```

For open source projects:
```markdown
| [Project Name](https://github.com/org/repo) | Brief description | ![Stars](https://img.shields.io/github/stars/org/repo?style=flat-square) |
```

### Requirements for Inclusion

- **Active:** App must be currently accessible and maintained
- **Relevant:** Must be primarily an AI companion, character chat, or relationship AI app
- **Established:** Must have been available for at least 1 month
- **Unique:** Must offer something distinct (not a pure clone/reskin of another listed app)
- **Accurate:** Pricing and features must be current

### What We Don't Accept

- General-purpose AI assistants (ChatGPT, Claude, Gemini) — these are productivity tools, not companions
- Apps that are defunct or consistently unavailable
- Pure spam/low-effort clone apps
- Apps with no English language support
- Paid listing requests — this is a community resource

### Updating Existing Entries

If an app has changed pricing, features, or platforms, feel free to submit a PR updating the information. Please note what changed in your PR description.

### Suggesting a New Section

If you think a new category is needed, open an Issue first to discuss before submitting a PR.

## Code of Conduct

- Be respectful in PRs and Issues
- Don't disparage other apps — factual comparisons are fine
- No promotional language — keep descriptions neutral and factual

## Questions?

Open an Issue if you're unsure whether something fits the list.
